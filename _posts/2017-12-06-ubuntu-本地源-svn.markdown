# ubuntu 本地源 svn #

### 安装 ###

	（1）sudo apt-get install subversion
	
	（2）sudo apt-get install apache2
	
	（3）sudo apt-get install libapache2-svn

其中subversion是SVN必须的，apache2和libapache2-svn是为了配置SVN服务器支持通过HTTP访问
